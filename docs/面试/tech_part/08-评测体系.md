## 2. Layer 1: Hard Rules

**文件**：`eval/layers/hard_rules.py`

```python
def run_layer1(report_md: str, case: dict) -> dict:
    """
    Layer 1 硬规则校验（无需 LLM）：

    通过规则检查报告的基本质量指标。
    快速筛选，不放过明显的失败案例。
    """

    checks = {}

    # Check 1: 报告非空
    checks["not_empty"] = len(report_md.strip()) > 0

    # Check 2: Section 数量
    section_headers = [line for line in report_md.split("\n") if line.startswith("## ")]
    checks["section_count"] = len(section_headers) >= 5
    checks["section_count_value"] = len(section_headers)

    # Check 3: Citation 数量
    citation_pattern = r"\[(\d+)\]"
    citations = re.findall(citation_pattern, report_md)
    checks["citation_count"] = len(citations) >= 3
    checks["citation_count_value"] = len(set(citations))  # 去重

    # Check 4: 无已知幻觉关键词
    hallucination_keywords = [
        "the study found that",
        "according to the paper",
        "it is well known that",
        "researchers have shown that",
    ]
    has_hallucination = any(
        kw.lower() in report_md.lower() for kw in hallucination_keywords
    )
    checks["no_hallucination_keywords"] = not has_hallucination

    # Check 5: 关键词召回率
    if "gold_keywords" in case:
        keywords_found = sum(1 for kw in case["gold_keywords"] if kw in report_md)
        checks["keyword_recall"] = keywords_found / len(case["gold_keywords"])

    # 综合判断
    all_passed = all(v if isinstance(v, bool) else v >= 0.5
                     for v in checks.values() if isinstance(v, (bool, float)))

    return {
        "pass": all_passed,
        "checks": checks,
        "report_snippet": report_md[:500],
    }
```

## 3. Layer 2: Grounding Metrics

**文件**：`src/eval/rag/metrics.py`

### 3.1 四层指标计算

#### Paper Retrieval Metrics（论文召回）

```python
def compute_paper_retrieval_metrics(
    predicted_ids: list[str],
    gold_ids: set[str],
    k_values: list[int] = [5, 10, 20],
) -> PaperRetrievalMetrics:
    """
    论文检索指标：

    - Recall@k：top-k 中包含 gold 论文的比例
    - MRR：首个 gold 论文的 reciprocal rank
    - NDCG@k：考虑排序质量的归一化折扣增益
    """

    results = {}

    for k in k_values:
        top_k = set(predicted_ids[:k])
        results[f"recall@{k}"] = len(top_k & gold_ids) / len(gold_ids) if gold_ids else 0.0
        results[f"ndcg@{k}"] = ndcg(
            [1 if pid in gold_ids else 0 for pid in predicted_ids[:k]], k=k
        )

    results["mrr"] = mrr([1 if pid in gold_ids else 0 for pid in predicted_ids])

    return PaperRetrievalMetrics(**results)
```

#### Citation Grounding Metrics（引用验证）

```python
def compute_citation_grounding_metrics(
    verified_claims: list[Claim],
    resolved_citations: list[Citation],
) -> CitationGroundingMetrics:
    """
    Citation Grounding 指标：

    - resolution_rate：引用 URL 可达率
    - support_rate：claim 被 evidence 支持率
    - abstention_rate：主动弃权率
    - tier_a_ratio：Tier A 引用占比
    """

    total_citations = len(resolved_citations)
    reachable = sum(1 for c in resolved_citations if c.reachable)
    resolution_rate = reachable / total_citations if total_citations > 0 else 0.0

    total_claims = len(verified_claims)
    supported = sum(1 for c in verified_claims if c.overall_status == "supported")
    partial = sum(1 for c in verified_claims if c.overall_status == "partial")
    abstained = sum(1 for c in verified_claims if c.overall_status == "abstained")

    support_rate = (supported + partial * 0.5) / total_claims if total_claims > 0 else 0.0
    abstention_rate = abstained / total_claims if total_claims > 0 else 0.0

    tier_a = sum(1 for c in resolved_citations if c.source_tier == SourceTier.A)
    tier_a_ratio = tier_a / total_citations if total_citations > 0 else 0.0

    return CitationGroundingMetrics(
        resolution_rate=resolution_rate,
        support_rate=support_rate,
        abstention_rate=abstention_rate,
        tier_a_ratio=tier_a_ratio,
    )
```

### 3.2 核心指标公式

| 指标 | 公式 | 含义 |
|------|------|------|
| **Recall@k** | \(\frac{|Pred_{1:k} \cap Gold|}{||Gold||}\) | top-k 召回率 |
| **MRR** | \(\frac{1}{\min(pos\ of\ first\ gold)}\) | 首个 gold 排名 |
| **NDCG@k** | \(\frac{DCG@k}{IDCG@k}\) | 排序质量（1=完美） |
| **Resolution Rate** | \(\frac{\#reachable\ citations}{\#total\ citations}\) | 引用可达率 |
| **Support Rate** | \(\frac{\#supported + 0.5 \times \#partial}{\#total\ claims}\) | claim 支撑率 |
| **Tier A Ratio** | \(\frac{\#TierA\ citations}{\#total\ citations}\) | 权威来源占比 |

### 5.2 Metric 含义解析

| Metric | 含义 | 目标值 |
|--------|------|--------|
| **Recall@10** | 召回率，衡量检索广度 | ≥ 0.80 |

| **Support Rate** | Claim 支撑率，衡量内容可信度 | ≥ 0.70 |

### 5.3 SciFact 数据集的 Benchmark 性质

SciFact 是一个**学术 claim verification benchmark**，特点：

1. **真实科学声明**：由领域专家撰写，非人工伪造
2. **粒度细**：claim-level 而非 document-level
3. **标注丰富**：每个 claim 配套 gold evidence + support type
4. **难度适中**：需要模型理解科学论文并判断 claim 是否被支持

**Baseline 参考**（来自 SciFact 官方）：
- Random baseline: ~20% resolution rate
- BM25: ~50% resolution rate
- Neural reranker: ~70% resolution rate

## 6. Eval Runner

**文件**：`eval/runner.py`

```python
def run_eval(
    cases_path: str = "eval/cases.jsonl",
    layer: int = 1,
    out_dir: str = "eval/runs",
) -> dict:
    """
    评测执行器：

    1. 加载 cases
    2. 对每个 case 执行 generate_literature_report
    3. Layer 1: 运行 hard_rules
    4. Layer 2: 运行 grounding metrics（需要 LLM）
    5. 保存结果到 eval/runs/
    """
    cases = load_cases(cases_path)
    results = []

    for case in cases:
        case_id = case["id"]
        report_md = generate_literature_report(arxiv_url_or_id=case.get("input"))

        l1 = run_layer1(report_md, case)

        result = {
            "id": case_id,
            "layer1": l1,
            "report_snippet": report_md[:300],
        }
        results.append(result)

    # 保存结果
    os.makedirs(out_dir, exist_ok=True)
    out_file = os.path.join(out_dir, f"run_{datetime.now().isoformat()}.json")
    with open(out_file, "w") as f:
        json.dump(results, f, indent=2)

    # 统计通过率
    passed = sum(1 for r in results if r["layer1"]["pass"])
    summary = {"total": len(results), "passed": passed, "pass_rate": passed / len(results)}
    return summary
```

## 7. 评测优势总结

| 优势 | 实现方式 |
|------|---------|
| **Layer 1 无 LLM 依赖** | Hard Rules 快速执行，不消耗 API quota |
| **分层递进** | L1 快速过滤 → L2 精细验证，不浪费资源 |
| **SciFact 标准化** | 使用学术 benchmark，结果可对比 |
| **NDCG/MRR 全面** | 衡量召回率 + 排序质量 + 排序位置 |
| **Claim-level 评估** | 比 document-level 更精细，能发现具体问题 |
